# Incident Report

## Incident Summary
Suspicious activities were detected including brute force login attempts, unauthorized access, and file deletion.

---

## Affected Systems
- Domain environment
- File storage system

---

## Findings
- Multiple failed login attempts followed by successful login
- Sensitive file accessed and deleted
- Guest account access observed

---

## Actions Taken
- Recommend password reset for affected accounts
- Disable guest account access
- Review access logs and permissions
- Implement account lockout policy

---

## Recommendation
Strengthen authentication controls and monitor login attempts.
