# 🛡️ SOC Analyst Detection Lab

## 📌 Overview
This project simulates a Security Operations Center (SOC) workflow including log analysis, threat detection, alert generation, and incident reporting.

---

## 🎯 Objectives
- Analyze logs to detect suspicious activity
- Identify potential security incidents
- Generate alerts based on findings
- Document incident response actions

---

## 🛠️ Tools Used
- Wireshark
- CyberChef
- Windows / Linux Logs
- Excel (Log Analysis)

---

## 📂 Project Structure
- Logs: Raw system/network logs
- Analysis: Investigated findings
- Alerts: Triggered alerts
- Reports: Incident reports
- Screenshots: Evidence of analysis

---

## 🚨 Outcome
Demonstrates hands-on SOC analyst skills including detection, analysis, and incident response.
